# Softwaredesign – Python Übungen

Dieses Repository enthält meine Lösungen zu den Softwaredesign-Übungen (Selfstudy B–E).
Die Aufgaben umfassen NumPy-Simulationen, pandas-Datenverarbeitung, OOP (Predator-Prey-Modell), Decorators und Logging.

## Ausführung

Abhängigkeiten installieren:

```bash
pdm install
```

Jupyter starten:

```bash
pdm run jupyter notebook
```
